# 🏥 Saúde Humanizada

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

Este projeto foi desenvolvido com o objetivo de criar uma interface web funcional e visualmente acolhedora, focada na temática de atendimento humanizado na área da saúde.

## Sobre o Projeto

Saúde Humanizada é um projeto extensionista realizado na Universidade Presbiteriana Mackenzie. A aplicação foca na construção de uma interface front-end utilizando tecnologias fundamentais da web para exercitar conceitos de design, acessibilidade e estruturação de documentos.

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de todas as páginas e seções.
* **CSS3:** Estilização visual, layouts e responsividade para garantir uma boa experiência em diferentes dispositivos.

## Como Visualizar

Como este é um projeto estático, você pode visualizá-lo facilmente:

1.  Clone o repositório:
    ```bash
    git clone [https://github.com/stephanievitoria/saude-humanizada-web.git](https://github.com/stephanievitoria/saude-humanizada-web.git)
    ```
2.  Aceda à pasta do projeto:
    ```bash
    cd saude-humanizada-web/"NOVA Saúde Humanizada"
    ```
3.  Abra o arquivo principal (geralmente `index.html`) no seu navegador.

## 📂 Estrutura de Pastas

* `NOVA Saúde Humanizada/`: Contém os arquivos de desenvolvimento (HTML e CSS).
* `README.md`: Documentação do projeto.
